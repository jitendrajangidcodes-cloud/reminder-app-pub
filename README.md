# Reminder — public site

Public homepage and privacy policy for the **Reminder** Android app
(`com.pnsjy.reminder_app`), served via GitHub Pages.

- Homepage: https://jitendrajangidcodes-cloud.github.io/reminder-app-pub/
- Privacy policy: https://jitendrajangidcodes-cloud.github.io/reminder-app-pub/privacy.html

This repo exists only to host the public pages required for Google OAuth
consent-screen verification. The app source lives in a separate private repo.
